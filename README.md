# 📦 Sistema de Cadastro de Produtos

Sistema desenvolvido em **Python** para realizar o cadastro e gerenciamento de produtos através do terminal, utilizando conceitos de programação e organização de dados.

## ✨ Funcionalidades

- Cadastrar produtos
- Listar produtos cadastrados
- Buscar produto
- Atualizar informações
- Remover produtos
- Validação de dados
- Interface via terminal

## 🎯 Objetivo

Este projeto foi desenvolvido para praticar conceitos fundamentais de Python, como:

- Variáveis e tipos de dados
- Estruturas de decisão e repetição
- Funções
- Manipulação de listas e dicionários
- Organização de código em módulos
- Persistência de dados
- Boas práticas de programação
